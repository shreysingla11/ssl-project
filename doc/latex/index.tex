 \hypertarget{index_autotoc_md4}{}\doxysection{About The Project}\label{index_autotoc_md4}
A Web based plagiarism detection tool that allows authenticated users to upload testfiles and displays the pairwise similarity results in the form of graphical visualizations. Specialized for c++ codes, the tool is capable of autodetecting base file snippets. ~\newline
 The Project has Four Parts\+:
\begin{DoxyItemize}
\item Core Logic\+: This part contains a parser and a k-\/gram modelling algorithm which is used to compute pairwise similarity for a given bunch of files
\item Frontend\+: Built using Angular Web Framework
\item Backend\+: Used Django Rest Framework to manage user data, batches and source code files ~\newline

\item Terminal Client\+: A basic command line interface for the Red\+Plag API developed using Node\+JS. Go to the respective folders for more detailed documentation.
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysection{Features}\label{index_autotoc_md5}

\begin{DoxyItemize}
\item k-\/gram model to represent code files and compute pair-\/wise similarity
\item Extensively tested and specialised for C++ and generalised to any arbitrary language
\item Autodetection of base file code snippets
\item Scatter Graph Visualization for Similarity Matrix
\item Dynamic Graph based on threshold set by user
\item Web Login and Register Setup
\item Command line Interface/ Terminal Client for our web service
\item File upload/download interface in Web\+App/\+Terminal Client
\item Implemented a rest API in Django using Django Rest Framework to manage userdata, batches and source code files
\item User profile page
\item Change password functionality
\item Detailed documentation for each part
