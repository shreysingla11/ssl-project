\item \#\#\# Login \`{}\`{}\`{} \$ login \`{}\`{}\`{} Login using your username, password and organisation passcode. This creates an authorization token which you include in subsequent requests.
\item \#\#\# Create \`{}\`{}\`{} \$ create \`{}\`{}\`{} Create a new batch of code files. The user needs to be logged in for this. You need to have all files in a single folder.
\item \#\#\# Batch listing, retreiving and deletion \`{}\`{}\`{} \$ batch \mbox{[}-\/d,--delete\mbox{]} \mbox{[}uid\mbox{]} \`{}\`{}\`{} Prints the list of batches or download result of batch with ID = uid. ~\newline
FLAG -\/d,--delete Deletes the batch with specified uid
\item \#\#\# Logout

\`{}\`{}\`{} \$ logout \`{}\`{}\`{} Logout from Red Plag server and dete the generated token
