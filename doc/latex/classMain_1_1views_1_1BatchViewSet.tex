\doxysubsection{Detailed Description}
This is the class controlling the authenticated /rp/batch endpoint. 

This viewset supports GET,POST,DELETE requests to perform CRUD operations on batches~\newline
 ~\newline
 GET /rp/batch Returns list of current user\textquotesingle{}s batches~\newline
 GET /rp/batch/\+:id Returns detail of batch with ID = id~\newline
 POST /rp/batch/ Creates a new batch for the current user~\newline
 DELETE /rp/batch/"{}\+:id Deletes the batch with ID=id if it belongs to the current user
