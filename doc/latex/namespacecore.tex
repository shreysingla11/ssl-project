\doxysubsection{Detailed Description}
This module describes the logic used to find plagiarism for code files. 

We have used k-\/grams models for computing similarity scores. Note that the logic autodetects Basefile efficiently and neutralises it\textquotesingle{}s affect. Hence giving an option to add base file snippet is redundant ~\newline

The core functionality is defined in this function. 

It first removes single-\/line and multiline comments, Identifies special characters, tokenises the code and stores kgrams of each code file. Next it identifies kgrams which are common to most of the files (They represent our base code file). So, we take each pair of files and calculate the k-\/grams which are common to the files but not common to most of the files. They form our suspicious set and will be used to calculate a score lying in 0-\/1, a metric which quantifies the probability that the pair is plagiarised.
